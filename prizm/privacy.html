#] PRIVACY ;]


Very simple privacy policy.

#] What's Stored

  • Reaction Roles
    • Emoji string [<:emoji_name:123456789012345678>]
    • Role ID [role name is NOT stored]
    • Message ID [message content is NOT stored]
  • Guild info
    • Guild ID
    • Guild name
  • Starboard
    • Message ID [message content is NOT stored]
  • Twitter
    • Tweeters
    • Channel IDs [channel names are NOT stored]
  • Chat bot
    • Message content, anonymously
    • User temporarily, used for context. Your username is associated with PRIZM ;]'s last message, never your own
  • Polls
    • Message ID [message content is NOT stored]
    • User ID to prevent spamming the up/downvote buttons. Your user ID just assigns your vote. User name is NOT stored.

#] How to delete data

All your data is deleted once PRIZM ;] leaves your server. When PRIZM ;] rejoins, you have to set everything back up

#] How can I see my data

Currently, I do not have a command allowing you to download your data. However, you can be sure that I'm not collecting too much

#] Where does my data go

It stays on PRIZM ;]'s hard drive. No data is sent anywhere, even for personalized advertising or whatever those nasty bots do.
Your data is used to ensure proper function of the bot, it's really only preferences and stuff.
If you want more information, feel free to email me with any questions.

IMPORTANT NOTICE PRIZM ;] is currently being rewritten to use /commands, so this privacy policy is subject to change to be more transparent about what data is being collected. Frankly, PRIZM ;] has so many commands that I forget what commands do what, but I assure you all data is anonymous other than server ID and server name. After the rewrite, server ID and server name won't be stored either.
