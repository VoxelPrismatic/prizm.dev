#] Smart Washer/Dryer/Dishwasher


A wonderfully bullshit idea. Imagine if you will a machine in which you have to load used/dirty materials in person, then go into another room to pull out your phone, open an app and click "Run". You have to load your washer/dryer/dishwasher in person anyway, why not start it with a few clicks right then and there?

#] Smart Thermostat


Wtf is smart about it? That it has an expensive screen? That it can connect to the internet?
I tried a smart thermostat [nest eco thermostat] and my room frequently got to 25c when the A/C was set to 23c and wouldnt run the A/C until it detected movement in the room or something.
Fun fucking fact: the only thing that I found smart about that thermostat was that it could automatically run heating and cooling without having to touch it, which can easily be done with a basic one for an extra $10 or so.
I have a basic thermostat which is arguably smarter because you can set how many times you would like the heat or A/C to cycle per hour so nothing can short cycle and damage itself.
I will accept a "semi smart" thermostat which only needs internet access for a better UI with the app, but that should only be an addon to an existing basic thermostat.
