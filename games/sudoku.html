Key map
Key Action
Α Move cursor up
< Move cursor left
> Move cursor right
V Move cursor down
Enter Write to cell
*Only works with "Digit first" enabled
1-9 Select a number, write to cell if "Cells first" is enabled
BKSP, DEL, 0 Clear cell
SHIFT, *, . Toggle notes
CTRL, / Toggle cells first or digit first mode
N Generate a new board
Home Move cursor to the center of the board
End Move cursor to the top right of the board
F Fill in the board with notes. This will take a while since these are advanced notes
